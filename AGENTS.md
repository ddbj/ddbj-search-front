# AGENTS.md

## この文書の役割

- この文書は、エージェントがこのリポジトリで作業する際の追加指示をまとめたものです
- 人間向けの一般ルールや日常運用の正本は `README.md` と `docs/` を参照してください
- この文書には、エージェントが最初に知っておくべき最小限だけを残します

## コミニュケーション言語

- 原則として日本語を使います。
  - 過去に英語を優先していた経緯があるので、内部には英語のドキュメントも残っていますが、中期的には日本語化していきます。
  - ただし、コード内の1行程度のコメントに関しては英語で残します。

## Skills

- `docs/agents/skills` フォルダ以下にタスク別の詳細な指示をまとめています
  - 該当するタスクの指示がある場合に参照してください

## 最初の入口

- まずは `README.md` を読んで、プロジェクトの入口と主要ドキュメントを把握してください
- 詳細な docs 一覧は `docs/index.md` を参照してください
- 最初から全てのドキュメントを読む必要はありません。必要になったものだけを、状況に応じて参照してください

## その他

- 変更・作成したファイル全体の出力は不要です。
  - ファイルパスの一覧だけを出力してください。
