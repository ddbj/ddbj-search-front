# AGENTS.md

## この文書で扱うこと

- この文書では、エージェント向けの追加指示だけを扱う。
- 一般ルールや日常運用は `README.md` と `docs/` を基準にする。
- ここには、作業の入口として必要な最小限だけを残す。

## 最初に確認すること

1. `README.md` を読んで、プロジェクトの入口と主要ドキュメントを把握する。
2. 詳細が必要になったら `docs/index.md` から必要な文書だけを開く。
3. 外部ツールを使う作業では `docs/external-tool-operation.md` も確認する。

## コミュニケーション

- やり取りは原則として日本語で行う。
- 既存の英語ドキュメントが残っていても、新しく書く説明や報告では日本語を優先する。
- ただし、コード内の短いコメントは英語のままでよい。

## タスク別の指示

- `docs/agents/skills/` には、タスク別の詳細指示がある。
- 該当する指示がある場合は、その文書を参照してから作業する。

## 出力

- 変更・作成したファイルの全文は出力しない。
- ファイルパスの一覧だけを示す。
